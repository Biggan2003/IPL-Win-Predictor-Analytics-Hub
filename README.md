# 🏏 IPL Winner Predictor

এটি একটি মেশিন লার্নিং মডেল যা IPL ম্যাচের ফলাফল ভবিষ্যদ্বাণী করে।

## কিভাবে ব্যবহার করবেন?
1. বাম দিকের ড্রপডাউন থেকে **Team 1** ও **Team 2** সিলেক্ট করুন
2. **Toss Winner** ও **Venue** সিলেক্ট করুন
3. Submit বাটনে ক্লিক করুন

## মডেল সম্পর্কে
- ট্রেনিং ডেটা: IPL 2008-2024
- অ্যালগরিদম: Random Forest
- Accuracy: ~52%
